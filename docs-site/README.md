# c3x docs site

VitePress-powered. To work on the docs locally:

```bash
cd docs-site
npm install
npm run dev    # http://localhost:5173
```

Build:

```bash
npm run build
```

Static output lands in `docs-site/.vitepress/dist/`. Deploy via
GitHub Pages / Netlify / Vercel — the build is dependency-free at
runtime.

## File map

```
docs-site/
├── .vitepress/config.mts      # sidebar, nav, theme
├── index.md                   # landing page
├── guide/                     # narrative docs (concepts, first run)
├── integrations/              # one page per forge / CI tool
└── reference/                 # CLI / catalog / architecture / migration
```

The catalog reference at `reference/catalog.md` is generated by
`go run ./cmd/gen_catalog_doc > docs-site/reference/catalog.md` —
re-run after adding new resource TOMLs.
